# YT-clone

It aims to be YouTube clone to test software architecture, coding skills,
production features like monitoring, security etc.

Features:

- [ ] Upload video with title, description
- [ ] suggested videos, feed
- [ ] security + oauth2
- [ ] Clean architecture / ports and adapters / DDD domain objects
- [ ] Monitoring
- [ ] Messaging to e.g. analyze video content id
- etc

- [ ] some JPA / Hibernate
- [ ] some JOOQ
- [ ] grpc
- [ ] kibana 
- [ ] grafana
- [ ] microservices, separate them, kubernetes?
